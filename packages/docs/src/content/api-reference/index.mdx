# API Reference

Reference documentation for the Revideo packages.

## Rendering

- [`@revideo/renderer`](/api-reference/renderer/renderVideo) — render projects
  to video from Node.js with `renderVideo()` and `renderPartialVideo()`.
- [`@revideo/ffmpeg`](/api-reference/revideo-ffmpeg/concatenate-media) —
  helpers for concatenating media and merging audio with video.
- [`@revideo/player-react`](/api-reference/player-react/player) — the React
  component for previewing projects in the browser.

## Animation Libraries

- [`@revideo/core`](/api-reference/core) — the animation engine: [flow
  control](/api-reference/core/flow), [signals](/api-reference/core/signals),
  [tweening](/api-reference/core/tweening),
  [transitions](/api-reference/core/transitions) and
  [types](/api-reference/core/types).
- [`@revideo/2d`](/api-reference/2d) — the 2D renderer and its
  [components](/api-reference/2d/components) like
  [`Rect`](/api-reference/2d/components/classes/Rect),
  [`Txt`](/api-reference/2d/components/classes/Txt) and
  [`Video`](/api-reference/2d/components/classes/Video).
