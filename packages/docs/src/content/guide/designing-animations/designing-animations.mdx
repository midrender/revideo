# Motion Canvas

Revideo was forked from [Motion Canvas](https://github.com/motion-canvas/motion-canvas) - therefore,
both projects largely share the same API for designing animations. If you want
to learn how to build animations, you should read through the
[Motion Canvas Guide](/animations/flow). API changes that are
specific to Revideo (such as the addition of
[Rive animations](/guide/designing-animations/rive-animations)) or relevant for Revideo's use cases, are
listed in the following pages.

If you like to learn from examples, you can check out our
[Code Snippets](/code-snippets/streaming-text), where you can find code examples of
common animations, such as [streaming text](/code-snippets/streaming-text) or
[moving objects](/code-snippets/moving-objects).
